# Bidirectional Searchh Algorithm

FInal Project for CIS 350 - Advanced Data Structures and Algorithms

## Overview

Implementation of a bidirectional searching algorithm to demonstrate basic core concepts.

## Features

- C++
- bidirectional search algorithm
- unordered mapping techniques
- queue (STL)
- linked list traversal techniques

## Purpose

The purpose of this project was to apply preset conditions to demonstrate proper function of the bidirectional search algorithm. 
More advanced uses of this algorithm would be for GPS routing systems, robotics, video game AI, and network communication where finding an efficient route between two points is necessary.
These systems may require the use of weighted graphs, massive datasets, real-time obstacle detection, or heuristic optimizations
